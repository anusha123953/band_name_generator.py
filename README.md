# band_name_generator.py
My first Python mini project. I learned how to run Python programs in PyCharm and take user input. This simple program asks for the city you grew up in and your pet’s name, then generates a fun band name by combining both inputs. Example output: MANGALORE RUBY.
CODE:
print("Welcome to the Band Name Generator.")
city = input("What's the name of the city you grew up in?\n")
pet = input("What's your pet's name?\n")
print("Your band name could be " + city + " " + pet)
